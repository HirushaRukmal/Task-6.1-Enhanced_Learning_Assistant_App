# Backend API for Enhanced Learning Assistant App

## Setup
1. Run `npm install`
2. Copy `.env.example` to `.env`
3. Add your real API key to `.env`
4. Run `npm start`

## Endpoints
- POST /llm/hint
- POST /llm/explain
- POST /llm/summary
- POST /llm/flashcards

The API key is stored in environment variables and is not exposed in the Android app.
